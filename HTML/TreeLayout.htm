Big tree

Demonstrates the tree layout. Click Ctrl-Shift-A to add a random node to the diagram, it'll be automatically layed out in the tree. Note that the layout starts with node zero and can cause problems if this node is not part of the tree. This is not a bug, the node considered as the root of the tree should be selected by the user in general. A generic graph has no root, so you need to point it out. See the code behind the example for more.
