Readme Notes for the Summer 2005 release:

  • This is still an intermediate release. Final release planned for end September (for my birthday) whereafter I'll focus on undo/redo, grouping and VS2005/Net2/Avalon. I guess version 3 will be very complete in terms of features.
  • You will notice in the code that for the first time I've added the code of the Neon library, don't expect too much though. I included some samples to highlight some of the features, the best place to look is the Cobalt application however since Neon was written in function of the Cobalt application.
  • NDoc compile graphlib API documentation
  • The new WordPress Netron site is being rewritten from scratch; cleaner, more streamlined, focused and I dropped obsolete stuff. This is the reason there is very little activity on the current one and the delayed release.
  • See the changelog if you wish to upgrade existing code or applications
  • Lots of new additions in this preview, highlights are:
    • cut/copy/paste
    • copy/paste from and to image format
    • dragdrop images from file-system
    • significantly changed the control-designer of the graph control with initial help for the novices. You can now dragdrop the control on a blank form and press F5 with a working graph-control and the basic shapes. Hints on how to connect events or add custom shapes...it should be really easy now.
    • automata stuff re-integrated, making the (old) Automatron app obsolete. I introduced control widgets in this context, they will be more general in next release.
    • SVG and HTML export
    • the collection of sample diagrams, put a lot of effort in showing all shapes and possibilities.
    • re-written z-rendering
    • Bezier serialization; see the mindmap sample
    • new set of flowcharting shapes
    • new hypershapes
    • ambient properties of the graph control are now serialized
    • create image-shape from existing sub-diagrams: you can create and save your own shapes in two clicks now!
    • lots more little things everywhere
  • Thanks for downloading this software and all your support via mail/forum: you make the difference!

 

Enjoy, Francois.
