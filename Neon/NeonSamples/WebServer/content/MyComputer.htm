Computer browser
 

This is an example of a custom location resolver. Resolvers are HTTP request handlers similar to HTTP modules or filters. The computer browser resolver allows you to access files from your disk using the registry to find out the MIME type and its associated applications.

For example, if you have a file named 'something.txt' on 'c:\temp' you could access it via

http://localhost:8080/MyComputer/c:/temp/something.txt

Of course, this can be accessed directly via 'c:\temp\something.txt' but this would bypass an eventual pre-processing from the webserver. This method is used in the image browser to filter out images on any given location.
