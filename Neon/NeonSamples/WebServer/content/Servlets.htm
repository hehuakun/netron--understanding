Servlets
 

Servlets are C# classes inheriting from the ServletPageBase class. They extend the functionality of the webserver and can be plugged into the server via the application configuration. The server comes with several examples showing how to write a servlet:

  • the image browser
  • the 'My computer' browser
  • XML display with server processed XSL
  • picture cropping
