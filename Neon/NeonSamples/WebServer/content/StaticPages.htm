Static pages
 

Static pages are files in some directory specified in the webserver's configuration and resolved by the content location resolver, which is just another HTTP module.

You can customize the location resolver in various ways to handle different file types or make appropriate changes to the file before display.

The pages of this Xeon overview are located in the 'content' directory of this application and are static pages.
