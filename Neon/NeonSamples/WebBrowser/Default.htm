Welcome to the Neon multi-tab browser demo

Click here to call a custom method in the main application. The CustomMethod resides in the Mediator class and it allows you to access the full power of the CLR from within an HTML page. In this example the custom method will load a new browser tab and navigate to the Netron homepage.

You can modify the style of the tabs via an extender object in the Neon library:

  • default style
  • Neon style

Note that all tab-pages will close before the UI is set to the new style. This is another example of inter-communication between browser document and the host (form).
